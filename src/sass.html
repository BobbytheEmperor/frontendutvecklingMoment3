Att börja arbeta med Sass

Som med allt nytt börjar det här momentet med ett "Vafan, hur får jag det här att fungera?" Arbetet med SASS, som ska vara smidigt och bra att använda för snyggare layout, framförallt kring arbete med färg, är i det första skedet trassligt. Steg ett är att se till att allt fungerar som det ska innan för mycket arbete läggs på uppgiften för Moment 3 i Frontend-utveckling. I skrivande stund fungerar endast HTML förutom att skapa kod för HTML är en prioritet att se till att alla bitar för Moment 3 fungerar.
